# MyPetCare-Smart-Companion-Animal-Care-System
MyPetCare adalah aplikasi berbasis web yang dirancang untuk mendukung pengelolaan perawatan hewan peliharaan secara terintegrasi dalam satu platform. Aplikasi ini dikembangkan menggunakan arsitektur Model–View–Controller (MVC) dan paradigma Object-Oriented Programming (OOP) dengan Java Servlet sebagai backend, Apache NetBeans sebagai IDE, serta MySQL sebagai basis data.
MyPetCare melibatkan tiga peran pengguna utama, yaitu pemilik hewan, dokter hewan/klinik, dan admin. Pemilik hewan dapat mengelola data hewan peliharaan, memantau pertumbuhan dan riwayat kesehatan, melakukan booking layanan klinik, menggunakan layanan penitipan hewan, serta berbelanja kebutuhan hewan melalui fitur marketplace. Aplikasi ini juga menyediakan fitur chat dua arah antara pemilik hewan dan pihak klinik untuk konsultasi, konfirmasi layanan, dan tindak lanjut perawatan secara terdokumentasi.
Selain itu, MyPetCare dilengkapi dengan peta interaktif untuk menampilkan lokasi klinik terdekat, pengingat otomatis jadwal konsultasi, serta tracker pertumbuhan hewan untuk mencatat perkembangan dan aktivitas medis secara berkala. Dengan fitur-fitur tersebut, MyPetCare berperan sebagai asisten digital yang membantu pemilik hewan menjalankan perawatan yang lebih sistematis, aman, dan efisien melalui satu platform terintegrasi.
